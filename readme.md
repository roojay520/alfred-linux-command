# alfred-linux-command

> Linux 命令大全搜索工具

## Install

```
$ npm install --global alfred-linux-command
```

*Requires [Node.js](https://nodejs.org) 4+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*


## Usage

In Alfred, type `lc`, <kbd>Enter</kbd>, and your query.
