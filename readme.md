# alfred-linux-command

> Linux 命令搜索工具

## Install

```
$ npm install --global alfred-linux-command
```

*Requires [Node.js](https://nodejs.org) 10+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*

## Usage

In Alfred, type `lc`, and your query.
